# Hotel Booking Cancellation Analysis

## 프로젝트 소개

이 프로젝트는 호텔 예약 데이터를 분석하여 예약 취소에 영향을 미치는 주요 요인을 파악하고, 이를 기반으로 취소율을 줄이기 위한 전략을 도출하는 것을 목표로 진행되었다.

단순한 데이터 요약이 아닌, 전처리 → 탐색적 분석 → 변수 간 관계 분석 → 비즈니스 인사이트 도출까지 전체 데이터 분석 흐름을 경험하고, 실제 서비스 문제 해결 관점에서 접근하였다.

---

## 프로젝트 수행 배경

호텔 예약 시스템에서 취소율은 매출 예측, 객실 운영, 인력 관리에 직접적인 영향을 미치는 핵심 지표다.  
예약이 실제 숙박으로 이어지지 않는 경우가 많기 때문에, 취소 패턴을 사전에 파악하는 것은 운영 효율을 높이는 데 매우 중요하다.

이 프로젝트는 다음과 같은 목적을 가지고 진행되었다.

- 데이터 기반으로 실제 비즈니스 문제를 해석하는 능력 확보
- 전처리부터 인사이트 도출까지 데이터 분석 전 과정 경험
- 단일 변수 분석을 넘어 복합 요인 분석 역량 강화
- 분석 결과를 실행 가능한 전략으로 연결하는 능력 향상

---

## 데이터 전처리

실제 데이터를 분석 가능한 상태로 만들기 위해 다음과 같은 전처리를 수행하였다.

- 결측치 처리  
  - children → 0으로 대체  
  - agent, company → 0으로 대체  

- 중복 데이터 제거  

- 이상치 탐색  
  - adults, children, babies 변수 확인  
  - 단순 제거가 아닌 데이터 맥락(단체 예약 등)을 고려하여 해석  

---

## 탐색적 데이터 분석 (EDA)

### 범주형 변수 분석

- 호텔 유형에 따른 취소율 차이
- 식사 유형별 취소율
- 시장 세그먼트별 취소율
- 유통 채널별 취소율
- 재방문 여부에 따른 취소율
- 객실 유형 및 배정 객실 차이에 따른 취소율
- 국가별 취소율

### 수치형 변수 분석

- lead time과 취소율 관계
- 특별 요청 수와 취소율 관계
- 전체 숙박 일수 및 평일/주말 숙박 일수와 취소율 관계
- 예약 변경 횟수와 취소율 관계
- 연도별, 월별 취소율 변화

### 변수 간 결합 분석

- 호텔 유형 + 특별 요청 수
- lead time + 특별 요청 수
- 예약 변경 횟수 + 숙박 일수
- lead time + 재방문 여부

단일 변수 분석으로는 확인할 수 없는 패턴을 복합적으로 탐색하였다.

---

## 주요 분석 결과

- 전체 예약 취소율: 약 27.7%

- City Hotel이 Resort Hotel보다 높은 취소율을 보임

- lead time이 길수록 취소율 증가  
  → 예약 시점과 실제 숙박 간의 간격이 클수록 취소 가능성 증가

- 특별 요청 수가 많을수록 취소율 감소  
  → 고객 관여도가 높을수록 예약 유지 가능성 증가

- 숙박 일수가 길수록 취소율 증가  
  → 일정 변화 리스크 증가

- 재방문 고객의 취소율이 더 낮음  
  → 고객 충성도와 취소율 간 관계 확인

- 특정 시기(7~8월)에 취소율 증가

- Online TA, TA/TO 채널에서 높은 취소율 발생

## 비즈니스 인사이트 및 개선 전략

분석 결과를 기반으로 다음과 같은 전략을 도출하였다.

- 환불 불가 또는 부분 환불 옵션 도입  
- 숙박 전 리마인더 알림 시스템 구축  
- 특별 요청 입력 유도 UX 개선  
- 신규 고객 대상 혜택 제공으로 취소 방지  
- 성수기 오버부킹 전략 활용  
- 취소율이 높은 채널에 대한 정책 재설계  

데이터 분석 결과를 실제 운영 개선 아이디어로 연결하는 데 초점을 맞추었다.

## 프로젝트를 통해 얻은 역량

### 데이터 전처리 능력
결측치, 중복값, 이상치를 직접 탐색하고 처리하면서 데이터 품질 관리 역량을 확보하였다.

### 탐색적 데이터 분석 능력
범주형 및 수치형 데이터를 구분하여 다양한 관점에서 데이터를 해석하고 의미 있는 패턴을 도출할 수 있게 되었다.

### 문제 해결 중심 데이터 해석 능력
단순한 수치 분석이 아닌, 왜 그런 결과가 나왔는지를 비즈니스 관점에서 설명하는 능력을 강화하였다.

### 변수 간 관계 분석 능력
여러 변수의 조합이 결과에 미치는 영향을 구조적으로 이해하고 분석할 수 있게 되었다.

### 데이터 기반 의사결정 능력
분석 결과를 실행 가능한 전략으로 연결하며, 데이터 분석이 실제 문제 해결에 어떻게 활용되는지 경험하였다.
